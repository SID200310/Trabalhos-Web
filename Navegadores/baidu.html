Baidu

A alternativa ao Chrome, mais social e oriental Baidu Spark Browser é um navegador desenvolvido com o Chromium, o mesmo motor usado no Google Chrome. Com a interface bem desenhada, muitos temas de personalização, integração com o Facebook e gerenciador de download para vídeos, Baidu é um ótima aposta para superar os browser tradicionais. Igual ao Chrome e com mais recursos O Baidu Spark Browser é um navegador curioso. Ele tem a estrutura do Google Chrome com alguns recursos extras bastante úteis e um novo layout. Parece pouco para um programa do buscador mais usado na China, mas, apesar do simplismo para descrevê-lo, o navegador é muito competente. Ao visualizar a interface, você notará uma grande barra lateral. Ali você pode acessar os sites favoritos, ver os downloads feitos, aumentar ou diminuir o tamanho da fonte das páginas e usar o Facebook. No caso da rede social, você usa o widget do programa para ver os posts publicados. Baidu Spark Browser permite configurar alguns gestos com o mouse para realizar ações específicas, como alternar, fechar ou abrir uma nova aba, trocar de janelas e muito mais. São dez gestos disponíveis até o momento, todos com explicação de como usar no menu " Configuração > Ferramentas". Um dos pontos fracos é não criar novas ações. Você fica refém do que já existe. Um dos pontos altos dos recursos é a opção "Media", que serve para você baixar vídeos do YouTube e sites similares. Outra função bem útil e vital nos dias de hoje é o recurso para diminuir o volume de todas abas. Pense naquela situação em que uma música irritante está sendo reproduzida e você não consegue descobrir de qual aba vem. Como no Chrome, o Baidu Spark Browser fornece acesso rápido a todos os sites visitados quando uma nova aba é aberta. Você ainda pode capturar telas, seja parte de uma página ou o conteúdo inteiro. Após a captura, a imagem pode ser editada no próprio editor do browser. Design personalizado O visual é idêntico ao Chrome, mas com cores mais infantis. Se o tema principal não agradar, você pode trocar por esquemas diferentes, encontrados por meio do botão que fica ao lado das abas de navegação. Os recursos espalhados na tela podem ser escondidos para você aumentar o campo de visão. A barra lateral, por exemplo, pode ficar oculta. Os atalhos dos recursos para baixar vídeos ou gerenciar os downloads estão bem localizados ao lado da barra de endereços e não incomodam. Uma aposta inesperada O Baidu foi uma aposta certa e inimaginada. O navegador está bem integrado com o Facebook e a barra lateral de atalhos não causa desconforto visual durante a navegação. Os recursos para baixar vídeos e capturar telas, sempre úteis, estão bem acessíveis. Os gestos automáticos do mouse, outra surpresa, responderam bem e melhoram a usabilidade do aplicativo. Sem sombra de dúvida, o Baidu Spark Browser é indicado para quem deseja largar os navegadores renomados. Você não vai perder nada em termos de qualidade ou velocidade de navegação. E, para ajudar, os recursos adicionais melhoram a experiência de navegação.
